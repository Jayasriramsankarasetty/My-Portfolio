# S. Jaya Sri Ram | Portfolio Website

This is the source code for my personal portfolio website, built using React.js.

## Features
- **Multi-Page Layout**: Home, About, Projects, and Resume sections.
- **Responsive Design**: Mobile-friendly layout using React-Bootstrap.
- **Dynamic Integrations**: GitHub contributions calendar.

## Getting Started
To run this project locally:

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Customization
You can easily customize this portfolio by editing the files inside `src/components/`. Personal information can be updated in `Home.js`, `AboutCard.js`, `Projects.js`, and `Footer.js`.
